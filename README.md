# NYX - AWS-native Satellite Telemetry Intelligence Platform

***

**NYX** is a cloud-native satellite telemetry intelligence platform designed to simulate telemetry flows, detect faults and suspicious behaviour, and expose operational intelligence through a multi-layered AWS data platform.

## Project Goals

This project aims to demonstrate:

- real-time telemetry ingestion
- secure cloud-native data platform design
- layered bronze / silver / gold S3 storage architecture
- anomaly detection using explicit rules and machine learning methods
- observability, alerting, and operational monitoring
- infrastructure as code (Terraform) and CI/CD discipline
